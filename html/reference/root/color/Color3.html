Color 3

Estructura de color de 3 canales (R, G, B)

Variables

Tipo Nombre Descripcion
unsigned char r Canal rojo
unsigned char g Canal verde
unsigned char b Canal azul

Funciones

Tipo Devuelto Prototipo Descripcion
Color3 __create_Color3 (unsigned char r, unsigned char g, unsigned char b) Crea un color con los parámetros r, g y b
Color3 __create_Color3 (int color) Crea un color mediante el parámetro de indice de colores integrados
unsigned char GetRed (Color3 color) Obtiene el canal rojo
void SetRed (Color3 color, unsigned char r) Establece el canal rojo
unsigned char GetGreen (Color3 color) Obtiene el canal verde
void SetGreen (Color3 color, unsigned char g) Establece el canal verde
unsigned char GetBlue (Color3 color) Obtiene el canal azul
void SetBlue (Color3 color, unsigned char b) Establece el canal azul

Descripción Detallada

Estructura de color de tres canales.

Contiene tres valores (R, G y B) y su valor puede oscilar entre 0 y 255, ambos incluidos.

Ejemplos

exampleModule.cc
#include "./Lucifer/common/lucifer.cc"

Lucifer::Color3 color;

void ExampleStart()
{
  color = Lucifer::COLORS3[Lucifer::LUCIFER_BLUE];
}

exampleModule.cc
#include "./Lucifer/common/lucifer.cc"

Lucifer::Color3 color;

void ExampleStart()
{
  color = {0, 128, 255};
}

exampleModule.cc
#include "./Lucifer/common/lucifer.cc"

Lucifer::Color3 color;

void ExampleStart()
{
  color.r = 0;
  color.g = 128;
  color.b = 255;
}
