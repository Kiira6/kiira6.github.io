Color 4

Estructura de color de 4 canales (R, G, B, A)

Variables

Tipo Nombre Descripcion
unsigned char r Canal rojo
unsigned char g Canal verde
unsigned char b Canal azul
unsigned char a Canal alfa

Descripcion Detallada

Estructura de color de cuatro canales.

Contiene tres valores (R, G, B y A) y su valor puede oscilar entre 0 y 255, ambos incluidos.

Ejemplos

exampleModule.cc
#include "./Lucifer/common/lucifer.cc"
#include "./exampleModule.cc"
 
...
Lucifer::Color4 color;
 
void ExampleStart()
{
	color = Lucifer::COLORS4[Lucifer::LUCIFER_BLUE];
}
 
...
exampleModule.cc
#include "./Lucifer/common/lucifer.cc"
#include "./exampleModule.cc"
 
...
 
Lucifer::Color4 color;
 
void ExampleStart()
{
    color = {0, 128, 255, 255};
}

exampleModule.cc
#include "./Lucifer/common/lucifer.cc"
#include "./exampleModule.cc"

Lucifer::Color4 color;
