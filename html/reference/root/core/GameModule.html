Estructura de módulo básico.

Es la estructura que utiliza Lucifer Framework para gestionar la modularidad de los programas.

Cada módulo está compuesto por las siguientes cuatro funciones:

  • Awake
  • Start
  • Update
  • Destroy

Todas son funciones opcionales y cada modulo puede tener cualquiera de ellas que necesite.

Ejemplos

exampleModule.cc
//  Example #1
 
void ExampleAwake()
{
    ...
}
 
void ExampleStart()
{
    ...
}
 
void ExampleUpdate()
{
    ...
}
 
void ExampleDestroy()
{
    ...
}
 
Lucifer::GameModule Example_Module =
{
    &ExampleAwake, &ExampleStart, &ExampleUpdate, &ExampleDestroy
};
exampleModule.cc
//  Example #2
 
void ExampleStart()
{
    ...
}
 
void ExampleUpdate()
{
    ...
}
 
Lucifer::GameModule Example_Module =
{
    NULL, &ExampleStart, &ExampleUpdate, NULL
};
main.cc
#include "./Lucifer/common/lucifer.cc"
#include "./exampleModule.cc"
 
...
 
void Lucifer::LoadModules()
{
    AddModule(Example_Module);
}
