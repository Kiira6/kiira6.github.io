Te damos la bienvenida a la web de referencia de la API Lucifer.

Con Lucifer podrás desarrollar con facilidad videojuegos 2D.

Lucifer funciona con las siguientes librerias externas:

  • EDK - ESAT Development Kit de ESAT.
  • ImGui - ImGui de Ocornut.

Lucifer contiene los siguientes módulos propios integrados:

  • Newton - Gestiona los componentes de física añadidos.
  • Baymax - Gestiona los robots añadidos al programa.

Además de código para la gestión de:

  • Color - Colores.
  • Containers - Bloques de datos.
  • Core - Núcleo principal de la librería.
  • Geometry - Estructuras geométricas.
  • Math - Módulo matemático.
  • Physics - Físicas.
  • Robot - Estructuras autogestionadas.
  • Transform - Estructuras de transformación.

Lucifer es un proyecto educacional.
